RULES:
1. Generate ONLY the document text — no commentary, no markdown formatting, no code blocks.
2. Use formal legal English appropriate for the startup stage specified.
3. All clauses must comply with the Indian Contract Act, 1872.
4. Include jurisdiction-specific language based on the Indian state provided.
5. Structure the document with numbered sections and clear headings.

5. Term and Duration of Confidentiality Obligations
6. Intellectual Property Rights
7. Return or Destruction of Information
